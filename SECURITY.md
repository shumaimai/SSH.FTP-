# セキュリティポリシー

Hashi は SSH 認証情報・sudo パスワードを扱い、リモートサーバーのファイル権限を
一時的に変更する機能(権限無視スイッチ)を持つソフトウェアです。
セキュリティ報告は歓迎します。

## 対象バージョン

最新リリース(`main` ブランチ)のみサポートします。

## 報告方法

脆弱性を見つけた場合は、**公開 Issue にせず**、GitHub の
[プライベート脆弱性報告](https://github.com/shumaimai/SSH.FTP-/security/advisories/new)
から報告してください(Security タブ → Report a vulnerability)。

特に以下は重大として扱います:

- 保存された認証情報(パスワード / パスフレーズ / sudo パスワード)の漏えい・平文化
- 権限無視スイッチが緩めた権限を復元し損ねる経路(ジャーナルの不備を含む)
- ホスト鍵検証(TOFU)のバイパス

## 免責

権限無視スイッチは「一時的に権限を緩めて直後に戻す」設計ですが、
利用は自分が管理権限を持つ(または明示的に許可された)サーバーに限ってください。
